#  **PHP简介**

PHP是一种广泛使用的开源服务器端脚本语言，主要用于 Web 开发。它可以嵌入到 HTML 中，特别适合用于创建动态网页和 Web 应用程序。

- PHP 文件的默认文件扩展名是 **.php**。

### **第一个php程序**

- PHP 脚本以 **<?php** 开始，以 **?>** 结束
- PHP 中的每个代码行都必须以分号结束。分号是一种分隔符，用于把指令集区分开来。
- PHP 中代码的缩进主要是为了提高代码的可读性和可维护性，可以适当缩进。
- 通过 PHP，有两种在浏览器输出文本的基础指令：**echo** 和 **print**
  - echo 和 print 区别:
    - echo - 可以输出一个或多个数据。
    - print - 只允许输出一个字符串。
    - **提示：**echo 输出的速度比 print 快。

```php
<?php
    // 输出一个字符串
    echo "Hello World!";
    print "我要学习 PHP!";
    // 输出多个类型的数据，echo输出张，用逗号对每个元素 进行分隔
    echo "景明",6666,"<br>";
?>
```

## **二、php变量**

**变量：用来存储数据的容器。**

### **PHP 变量规则：**

- 变量以 $ 符号开始，后面跟着变量的名称
- 变量名必须以**字母**或者**下划线**字符开始  
- 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ），不能包含空格   
- 变量是用于存储信息的"容器"
- 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）

### **PHP变量声明：**

```php
<?php
    // 变量存值 （注意：当赋一个文本值给变量时，需用在文本值两侧加上引号。）
    $name="张三"; 
    $age=18;
    // 变量存计算结果 （解题：食堂买来60棵大白菜，吃了56棵，又买来30棵，现在还有多少棵大白菜？）
    $num = 60-56+30;
    echo $name.$age.$num  // echo输出多个数据时，每个被输出的元素用.进行分隔  拼接
?>
```

## **三、数据类型**

在PHP 中变量可以存储不同的类型的数据，不同的数据类型可以做不一样的事情。

PHP 支持以下几种基础数据类型:

- String（字符串）

### **PHP 字符串**

**什么是字符串？**

**在编程中用引号包裹起来的一串字符是字符串数据。**

一个字符串是一串字符数据，就像 "Hello world!123"，就像 " **你好世界** "，都是字符串 。

**在php中字符串的表示形式？**

用单引号或者双引号包裹。

```php
<?php 
    $x = "Hello world!"; 
    echo $x;

    echo "<br>";     // 输出换行

    $x = 'Hello world!'; 
    echo $x;
    
    echo "<br>";     // 输出换行    
    
    // 字符串拼接
    $str1 = "phpStorm";
    $str2 = ".exe" ;
    // 要求将$str1与$str2拼接生成一个完整的字符串
    // 在php中可以通过 . 去拼接字符串
    $str3  = $str1.$str2;   // 如果拼接好的数据后期需要被用到，可以用变量保存
    echo $str3; 
?>
```

###  **PHP 整型**

### **PHP 数组**

**什么是数组？**

由多个值组成的，一个完整的数据序列，就是数组。

**数组有什么用？**

数组可以在一个变量中存储多个值。

**如何创建数组？**

在 PHP 中，array() 函数用于创建一个数组。

索引数组语法：array(值1, 值2, 值3...)

关联数组语法：array(键1=>值1 , 键2=>值2 , 键3=>值3 ...)

**如何访问索引数组中的值？**

数组中的每个值，电脑都会给其标序号，序号从0开始，这个序号我们称为索引。程序员通常通过下标访问数组的每个值。

索引取值语法：**数组名[索引]**

**如何访问关联数组中的值？**

关联数组中每个值会对应一个键。程序员通常通过键访问数组的值。

关联取值语法：**数组名[键]**

**在以下实例中创建了一个数组， 然后使用下标取值的方式，打印数组中的每个元素。**

```php
<?php
    // 索引数组：
    // 使用 array() 将 "张三"、"李四" 和 "王五" 这三个元素存储在一个数组变量 $user 中。
    $user = array("张三","李四","王五");
    //      下标：   0     1      2
    echo "数组：".$user[0]."、".$user[1]."、".$user[2];
    
    // 关联数组：
    // 关联数组使用自定义的键（这里是 "a" 和 "b"）来关联对应的值（这里是 "red" 和 "green"）
    $x = array("a" => "red", "b" => "green");
    // 关联数组可以让你通过指定的键快速访问其对应的值，例如，你可以使用 $x["a"] 来获取 "red" 这个值。
     echo $x["a"];
?>
```

## **四、控制结构**

### **PHP** **If...Else** **语句**

条件语句用于根据不同条件执行不同动作。

- **if 语句 - 在条件成立时执行代码。**

**语法：**

> **if **(判断条件1){
>
> ​	// 满足条件1 后执行的代码
>
> **}**

**如果当前值小于 18，下面的**实例将输出 "未成年哦！"：

```php
<?php 
    $age = 16; 
    if ($age < 18){ 
        echo "未成年哦！"; 
    } 
?>
```

- **if...else 语句 - 在条件成立时执行一块代码，条件不成立时执行另一块代码**

**语法：**

> **if** (判断条件1){
>
> ​	// 满足条件1 后执行的代码
>
> **}else if **(判断条件2){
>
> ​	// 满足条件2后执行的代码
>
> **}**

下面的实例：如果当前值小于 18，则将输出 "未成年哦！"，否则输出 " 成年啦！"：

```php
<?php 
    $age = 18; 
    if ($age < 18) { 
        echo "未成年哦！"; 
    }else{
        echo "成年啦！";    
    } 
?>
```

- **if...elseif....else 语句 - 在若干条件之一成立时执行一个代码块**

**语法：**

> **if** (判断条件1){
>
> ​	// 满足条件1 后执行的代码
>
> }**else if** (判断条件2){
>
> ​	// 满足条件2 后执行的代码
>
> **}**
>
> ......// 可以有更多的elseif...  
>
> **else**{
>
> ​	// 不满足上面所有条件才执行的代码
>
> **}**

下面的实例：如果当前值小于 18，则输出 "未成年哦！"，如果当前值等于 18，则输出 "刚好成年呀！"，否则输出 " 是个大人啦！"：

```php
<?php 
    $age = 18; 
    if ($age < 18) { 
        echo "未成年哦！"; 
    }elseif ($age == 18) {
        echo "刚好成年呀！";        
    }else{
        echo "是个大人啦！";    
    } 
 
```

### PHP Switch 语句

switch 语句用于根据多个不同条件执行不同动作。

> **switch** (表达式) {  
>
> ​    **case** **值1**:  
>
> ​        	 // 如果 表达式 等于 值1 时，执行的代码  
>
> ​       		**break;** // 可选，用于跳出switch语句  
>
> ​    **case** 值2: 
>
> ​       		 // 如果 表达式 等于 值2 时，执行的代码  
>
> 				**break;** // 可选，用于跳出switch语句  
>
> ​    ...... // 可以有更多的情况...  
>
> ​    **default**:
>
> ​        	// 如果没有任何case匹配时，执行的代码  
>
> **}**

**工作原理：**首先对一个简单的表达式 （通常是变量）进行一次计算。将表达式的值与结构中每个 case 的值进行比较。如果存在匹配，则执行与 case 关联的代码。代码执行后，使用 **break** 来阻止代码跳入下一个 case 中继续执行。**default** 语句用于不存在匹配（即没有 case 为真）时执行。

```php
<?php 
    $favcolor = "red"; 
    switch ($favcolor) { 
    case "red": 
        echo "你喜欢的颜色是红色!"; 
        break; 
    case "blue": 
        echo "你喜欢的颜色是蓝色!"; 
        break; 
    case "green": 
        echo "你喜欢的颜色是绿色!"; 
        break; 
    default: 
        echo "你喜欢的颜色不是 红, 蓝, 或绿色!"; 
    } 
?>
```

 

## **五、循环结构【while、for...each】**

### **PHP 循环 -** **While 循环**

while 循环将重复执行代码块，直到指定的条件不成立。

**语法：**

> **while** (条件){
>
> 要执行的代码;
>
> }

下面的实例首先设置变量 i 的值为 1 ($i=1;)。然后，只要 *i* 小于或者等于 5，while 循环将继续运行。循环每运行一次，i 就会递增 1：



```php
<?php
    $i=1;
    while($i<=5){
        echo "这个数字是 " . $i . "<br>";
        $i++;
    }
?>
```

### **PHP 循环 -** **Foreach 循环**

foreach 循环用于遍历数组。

**语法1：**

> foreach ($array as $value){
>
> ​    要执行代码;
>
> }

**每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。**

**语法2：**

> foreach ($array as $key => $value){
>
> ​    要执行代码;
>
> }

**每一次循环，当前数组元素的键与值就都会被赋值给 $key 和 $value 变量（数字指针会逐一地移动），在进行下一次循环时，你将看到数组中的下一个键与值。**

下面的实例演示了一个输出给定数组的值的循环：

```php
<?php 
    $x=array("谷歌","菜鸟","京东"); 
    foreach ($x as $value){ 
        echo $value . "<br>"; 
    } 
?>
```

下面的实例演示了一个输出给定数组键与值的循环：

```php
<?php 
    $x=array(1=>"谷歌", 2=>"菜鸟", 3=>"京东"); 
    foreach ($x as $key => $value) { 
        echo "key  为 " . $key . "，对应的 value 为 ". $value . "<br>"; 
    } 
?>
```

##  **六、函数**

函数是程序员定义好的，可以重复使用的代码块

PHP 的真正威力源自于它的函数。在 PHP 中，提供了超过 1000 个内建的函数。在本章中，我们将为您讲解如何创建自己的函数。如要在页面加载时执行脚本，可以把它放到函数里。函数是通过调用函数来执行的。你可以在页面的任何位置调用函数。

### **PHP** **初识函数**

**基础语法：**

> function  函数名(){
>
> ​	// 要执行的代码
>
> }

PHP 函数准则：

- 函数的名称应该提示出它的功能
- 函数名称以字母或下划线开头（不能以数字开头）

**以下实例是一个简单的函数，在其被调用时能输出我的名称：**

```php
<?php
    //函数定义 
    function printName() { 
        echo "景明"; 
    } 

    echo "我的名字是 "; 
    printName();     //函数调用 
?>
```
